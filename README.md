# 🚀 JVN Ganesh — Personal Portfolio Website

> **Live Site:** [jvnganesh.github.io/My_Portfolio_Website](https://jvnganesh.github.io/My_Portfolio_Website)

A high-performance 3D portfolio website built with Three.js and GSAP, featuring an interactive galaxy particle system, scroll-triggered animations, and a full cosmic/space visual theme.

---

## ✨ Features

- **Interactive 3D Galaxy** — 9,000-particle spiral galaxy with real-time mouse parallax
- **Floating Wireframe Objects** — Icosahedrons and octahedrons drifting in 3D space
- **Custom Animated Cursor** — Neon ring + dot with hover morphing
- **Typewriter Effect** — Cycles through 6 professional roles
- **Scroll Animations** — GSAP ScrollTrigger on every section
- **3D Card Tilt** — Mouse-reactive perspective tilt on project/research cards
- **Stat Counters** — Animated number counters on scroll
- **Certifications Marquee** — Infinite-scroll certification carousel
- **Fully Responsive** — Mobile, tablet, and desktop

---

## 🛠 Tech Stack

| Layer | Technology |
|---|---|
| 3D Graphics | Three.js r128 |
| Animations | GSAP 3.12 + ScrollTrigger |
| Fonts | Orbitron + Space Grotesk (Google Fonts) |
| Deployment | GitHub Pages |
| Build | Vanilla HTML / CSS / JS — zero dependencies |
